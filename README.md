📊 圖形化控制中心 (GUI 面板)
一鍵呼叫： 遊戲內按下 V 鍵即可開啟半透明的控制面板。

可視化操作： 點擊按鈕即可開關各項功能，並能直接在面板上點擊 ✖ 快速移除黑名單物品。

數據統計大廳： 右側面板會顯示當前環境的實際拋竿掛機時間、總上鉤次數，以及精確到小數點的戰利品機率排行。面板內建「重置統計」按鈕，隨時重新計算你的漁獲！

📡 實體軌跡追蹤 (防干擾統計系統)
拋棄傳統且容易誤判的「背包比對法」，本模組採用獨家**「生成點實體雷達 (Spawn Entity Tracking)」**技術。

在收竿瞬間鎖定魚漂座標，精準捕捉從水中彈出的戰利品。即使你在擁擠的伺服器主城掛機，旁邊有玩家狂丟垃圾或鑽石給你，統計數據也絕對不會被污染，保證 100% 的準確率！

🌊 客戶端強制開放水域掃描
伺服器端修改了設定導致原版判定失效？沒關係！

模組內建客戶端暴力掃描器，每次拋竿會自動掃描魚漂周圍 5x4x5 (共 100 個方塊) 的物理環境，精準告訴你當前水池是否符合 Minecraft 原版「開放水域 (可釣寶藏)」的嚴格規定，並在經驗條上方顯示 ✅ 或 ❌。

🗑️ 智慧環境隔離黑名單
每個單人世界、每個多人伺服器（依據 IP）的「丟棄黑名單」與「統計數據」都是完全獨立存檔的。你在 A 伺服器設定的垃圾，絕對不會影響 B 伺服器。

釣到黑名單物品時，會在 0.25 秒內自動丟出背包。

🎣 貼心的基礎防呆機制
全自動釣魚： 咬鉤瞬間精準收竿，並帶有隨機延遲（25~54 Ticks）模擬真人操作，防反作弊偵測。

自動補貨釣竿： 當手上釣竿用壞時，會自動從背包中尋找備用釣竿並替換。

耐久防斷警告： 釣竿耐久度低於 10, 5, 3, 2, 1 時顯示紅色警告字樣。

水中防呆： 魚漂如果丟到陸地上，會自動收回重新拋竿。
