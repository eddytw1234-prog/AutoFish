自動釣魚模組 (AutoFish) By 閒著的賢者
一個為 Minecraft 1.20.1 Fabric 打造的智慧型自動釣魚模組。不僅能幫你全自動釣魚，還配備了釣竿防斷警告、自動替換、防呆偵測，以及最強大的 「按伺服器獨立」自動丟棄垃圾系統，讓你的掛機體驗無比順暢！

✨ 核心功能 (Features)
🎣 全自動釣魚： 咬鉤瞬間精準收竿，並帶有隨機延遲（25~54 Ticks）模擬真人操作。

🔄 自動補貨釣竿： 當手上釣竿用壞時，會自動從背包中尋找備用釣竿並替換至手中。

⚠️ 耐久度防斷警告： 當釣竿耐久度低於 10, 5, 3, 2, 1 時，會在經驗條上方（Action Bar）顯示顯眼的紅色警告，方便你及時更換有附魔的好釣竿。

🛡️ 水中防呆偵測： 魚漂如果不小心被丟到實體方塊（陸地）上，模組會自動收回重新拋竿，防止傻傻掛機。

🗑️ 智慧自動丟棄 (支援自定義物品)： * 釣到不想要的垃圾？直接丟棄！支援伺服器特製的彩色名稱物品。

資料絕對隔離： 每個單人世界、每個多人伺服器的「丟棄黑名單」都是獨立存檔的。你在 A 伺服器設定的垃圾，絕對不會影響你在 B 伺服器的設定。

📥 安裝說明 (Installation)
確保你已安裝 Fabric Loader (對應 Minecraft 1.20.1 版本)。

確保你的 mods 資料夾內有安裝 Fabric API。

下載本模組的 .jar 檔案，並放入 .minecraft/mods/ 資料夾中。

啟動遊戲！

🎮 按鍵與指令 (Usage)
預設快捷鍵
K 鍵： 自動釣魚總開關 (開啟/關閉)。

O 鍵： 水中防呆偵測開關 (開啟/關閉)。
(你可以在遊戲內的「選項 -> 控制 -> 按鍵設定」中自由更改這些按鍵)

快速設定指令
不想切換視窗改設定檔？直接在遊戲中拿著物品輸入指令：

/autofish trash add
👉 將主手拿著的物品加入當前伺服器/世界的自動丟棄黑名單。

/autofish trash remove
👉 將主手拿著的物品從當前伺服器/世界的自動丟棄黑名單中移除。

⚙️ 設定檔位置 (Configuration)
所有的自動丟棄名單都存放在以下路徑：
%appdata%\.minecraft\config\fishingautomation\

你可以在裡面找到針對不同伺服器與存檔獨立生成的 .txt 檔案（例如 trash_Server_mc.hypixel.net.txt 或 trash_World_生存世界.txt）。你可以直接用記事本打開它們進行手動編輯（一行一個物品名稱），修改後在遊戲中按 K 鍵重新開啟模組即可套用。

⚠️ 伺服器使用警告 (Disclaimer)
這是一個純客戶端 (Client-side) 模組，伺服器端不需要安裝即可運作。
