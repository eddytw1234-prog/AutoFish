自動釣魚模組 (AutoFish Ultimate)
專為 Minecraft Fabric 打造的智慧型自動釣魚輔助模組。

✨ 已擁有功能
全自動釣魚：精準咬鉤收竿，自帶隨機延遲模擬真人操作。

圖形化控制面板 (GUI)：視覺化開關功能、管理黑名單與查看統計數據。

無縫自動換竿：釣竿損壞時，自動從背包物理抽換備用釣竿，完美接住剛釣起的漁獲。

獨立環境黑名單 (自動丟棄)：單人世界與伺服器黑名單各自獨立。可透過 GUI 的 + 號按鈕快速將手中物品加入丟棄清單。

重釣檢測機制：當魚漂落到陸地、勾到實體（生物/掉落物）、或在水中超過 60 秒無反應時，自動收竿重拋防卡死。

開放水域掃描：拋竿自動掃描魚漂周圍 5x4x5 空間，判定是否符合原版「可釣寶藏」條件。

精準漁獲統計：只計算「真正掛機中」的時間，並即時排行戰利品機率與總數。

全方位防干擾與防偷系統：

漁獲防搶：利用背包快照比對，漁獲在半空中被別人吸走時觸發。

防惡意推擠：偵測並警告試圖與你發生碰撞的玩家。

防惡意勾取：偵測並警告試圖用魚漂將你勾走的玩家。

水中漂浮：偵測玩家是否掉入水中，並在水中時自動上浮。

(註：以上異常皆會在經驗條上方顯示紅字警告，並靜默寫入 errors 資料夾的每日日誌中，日誌保留 7 天自動清理。)

⌨️ 快捷鍵配置
V 鍵：開啟自動釣魚控制中心 (GUI 面板)

K 鍵：自動釣魚 總開關

O 鍵：重釣檢測 開關

(以上快捷鍵皆可於遊戲內的「選項 -> 控制 -> 按鍵設定」中自訂修改)
